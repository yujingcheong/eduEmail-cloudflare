# 🚀 项目概览 - 临时邮箱生成器

## 📸 界面预览

### 主界面
![临时邮箱生成器主界面](temp-email-frontend.png)

*简洁优雅的用户界面，一键生成临时邮箱地址*

### 功能特性
- ✅ **一键生成** - 点击按钮即可生成随机临时邮箱
- ✅ **实时接收** - 自动接收发送到临时邮箱的所有邮件  
- ✅ **内容查看** - 支持 HTML 和纯文本邮件显示
- ✅ **批量管理** - 可查看历史邮箱和批量删除
- ✅ **隐私保护** - 无需注册，不保存个人信息

## 🏗️ 技术架构

### 系统组成
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   前端界面      │    │   UniCloud      │    │   Cloudflare    │
│  (HTML/JS/CSS)  │◄──►│    云函数       │◄──►│  Workers+Email  │
└─────────────────┘    └─────────────────┘    └─────────────────┘
        │                       │                       │
        ▼                       ▼                       ▼
  用户操作界面           API逻辑处理              邮件路由处理
```

### 数据流向
1. **邮箱生成**: 前端 → UniCloud → Cloudflare API → 创建邮件路由
2. **邮件接收**: 外部邮件 → Cloudflare Email Routing → Workers → UniCloud → 数据库
3. **邮件查看**: 前端 → UniCloud → 数据库查询 → 返回邮件内容
4. **邮箱删除**: 前端 → UniCloud → Cloudflare API → 删除路由规则
